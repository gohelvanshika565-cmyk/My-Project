# JavaScript Basic - 20 Programs

## 📖 Project Overview
This project contains 20 basic JavaScript programs designed to help beginners understand fundamental JavaScript concepts such as variables, operators, conditional statements, loops, functions, arrays, and objects.

The programs are simple, easy to understand, and useful for learning the basics of JavaScript programming.

---

## 🎯 Objectives
- Learn JavaScript syntax and structure.
- Understand variables and data types.
- Practice conditional statements and loops.
- Work with functions and arrays.
- Improve problem-solving skills using JavaScript.

---

## 🛠 Technologies Used
- HTML
- JavaScript (ES6)
- Web Browser
